hexo出现spawn-failed错误

发表于 2021-05-14 分类于 Hexo ， hexo优化

使用hexo g -d更新的时候报spawn failed错误

上网找到解决方案

原因

因为git进行push或者hexo d的时候改变了.deplo_git文件下的内容

解决方案

  1. 删除.deploy_git文件夹;
  2. 输入git config --global core.autocrlf false
  3. 然后，依次执行： hexo clean hexo g hexo d
